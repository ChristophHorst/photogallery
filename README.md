photogallery
============

Android Flickr photo gallery app, still early in its development.

Android features used in this app:

* Activities
* Fragments
* GridViews
* Action Bars
* Intents
* HTTP and Background Tasks
* Networking
* Threads
* XML
* Loopers
* Handlers
* HandlerThreads
* Messages and Message Handlers
* Search and SearchViews
